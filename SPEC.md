# My Schedule — 要件メモ

## 目的
個人専用のスケジュールプランナー。**平日用**と**休日用**の2つのスケジュールを
**30分単位**で登録・編集できるようにする。

## 元の依頼内容（ユーザーより）
> このプロジェクトを私専用の平日用と休日用のスケジュールを30分単位で登録できるように
> してください。しかも、このセッションで確認できるようなシステムで、スマホからアクセス
> する予定にしてください。

## 要件
- [ ] 平日用スケジュール（30分単位）の登録・編集・削除
- [ ] 休日用スケジュール（30分単位）の登録・編集・削除
- [ ] スマホ（モバイルブラウザ）からアクセスできるレスポンシブUI
- [ ] このセッション中に動作確認できること
- [ ] 個人専用（認証は当面なし想定 / 後述の確認事項参照）

## 決定事項
1. **アクセス方法**: GitHub Pages で公開（固定URLにスマホからいつでもアクセス可）
2. **1日の表示時間帯**: 4:00〜24:00（30分単位＝40枠）
3. **データの保存先**: ブラウザの localStorage（端末ごと）
4. **入力する内容**: 予定タイトルのみ

## 技術方針
- フレームワーク不要の静的Webアプリ（HTML + CSS + JavaScript）
- データはブラウザの localStorage に保存（サーバ・DB不要、個人利用に十分）
- GitHub Pages で公開してスマホからアクセス（GitHub Actions で自動デプロイ）
